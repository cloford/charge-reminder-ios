# ChargeReminder

iOS app MVP for a bedtime charging reminder.

## Current scope

- SwiftUI 4-tab skeleton
- Local notification permission and scheduling
- Up to three reminder times
- Battery level and charging state display
- UserDefaults-based settings persistence

## Bundle ID

`com.cloford.chargereminder`

## Notes

This app is a reminder, not an alarm replacement. Battery checks happen while the app is active or returns to foreground.
